# Engineering

Skills for code work.

- **[typescript-expert](./typescript-expert/SKILL.md)** — Enforces senior-level strict TypeScript practices across the full stack: no `any`, branded ID types, Zod-based runtime validation with inferred types, explicit return types, discriminated-union error handling, and DRY utility types.
- **[dev-task](./dev-task/SKILL.md)** — Develop a Jira issue end-to-end: gather requirements (interviewing if thin), plan a breakdown for parallel work, get one approval, then develop via parallel sandboxed subagents, consolidate, run code review, and open a PR ready for review, linked back to Jira.
